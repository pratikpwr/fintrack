# Satoshi Font

Download the Satoshi font from: https://www.fontshare.com/fonts/satoshi

Place the following font files in this directory:
- Satoshi-Light.otf (weight: 300)
- Satoshi-Regular.otf (weight: 400)
- Satoshi-Medium.otf (weight: 500)
- Satoshi-Bold.otf (weight: 700)
- Satoshi-Black.otf (weight: 900)

The app will use system fonts as fallback until these files are added.
